## 📖 核心主題：RSA 非對稱加密
**RSA** 是現代數位安全（如 HTTPS、數位簽章）的基礎。其核心價值在於「加密」與「解密」使用不同的金鑰，即便加密方法公開，沒有私鑰也無法破解訊息。

### 1. 數學原理白話文
RSA 的安全性建立在**大數分解的困難性**：
* **容易的事**：把兩個質數相乘（例如 $61 \times 53 = 3233$）。
* **困難的事**：給你 $3233$，叫你找出它是哪兩個質數相乘（當數字變大到幾百位數時，電腦也算不出來）。
